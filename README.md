Purpose: Automation of job search and freelance projects. The app eliminates the need for manual site verification by collecting data on its own and notifying you of new ads in real time.

Key functions:
Background work with system tray placement.
A settings window with filtering by keywords, region, and other parameters.
An intelligent date parser that recognizes relative formulations ("yesterday at 2:30 p.m.", "3 hours ago", "just now") through regular expressions.
Pop-up notifications when suitable vacancies and projects are found.
Closing control: the "close" button hides the window, and full exit is via the tray menu.
